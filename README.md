# Quiz – Criatividade do Rei (2024) 

Este projeto foi desenvolvido em PHP com o objetivo de permitir que colegas participem de um quiz interativo sobre o livro **“Criatividade do Rei”**, utilizado na disciplina de **Gestão** no ensino médio.

## Objetivo

Facilitar a revisão do conteúdo do livro de forma dinâmica e colaborativa, promovendo o aprendizado através de perguntas e respostas.

## Como funciona

* O sistema roda em um **servidor local**
* Os participantes acessam pelo navegador (mesma rede)
* Cada usuário responde às perguntas do quiz
* O sistema registra e exibe os resultados

## Tecnologias utilizadas

* PHP
* HTML/CSS
* Servidor local (ex: XAMPP, WAMP)

## Como executar

1. Instale um servidor local (XAMPP, WAMP, etc.)
2. Coloque os arquivos do projeto na pasta `htdocs` (ou equivalente)
3. Configure o servidor para aceitar conexão entre outros coputadores
4. Inicie o servidor
5. Acesse via navegador:

   ```
   http://localhost/nome-do-projeto
   ```
6. Compartilhe o IP da sua máquina para que outros na rede possam acessar
